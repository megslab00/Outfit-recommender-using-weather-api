# Outfit recommender using weather api
A smart web-based outfit recommendation system that suggests suitable clothing based on real-time weather conditions. This project integrates a pre-trained deep learning model with a weather API to personalize fashion choices according to temperature, humidity, and weather type (e.g., sunny, rainy, cold).
